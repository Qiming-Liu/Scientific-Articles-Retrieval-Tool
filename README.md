### Front-end
```shell
# install dependence
yarn
# development
yarn dev
# build
yarn build
# run
yarn start
```
> yarn latest required

### Back-end
```shell
# install dependence
pip install -r requirements.txt
# run
python app.py
```
> python3.6+ required
