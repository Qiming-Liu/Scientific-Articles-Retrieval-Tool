### Front-end (Required yarn latest)
```shell
# install dependence
yarn
# development
yarn dev
# build
yarn build
# run
yarn start
```

### Back-end (Required python3.6+)
```shell
# install dependence
pip install -r requirements.txt
# run
python app.py
```
